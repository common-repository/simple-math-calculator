== Installation ==

**Normal installation**

1. Download the wp-simple-cal.zip file to your computer.
1. Unzip the file.
1. Upload the `wp-simple-cal` directory to your `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

That's it. You can use the the plugin by using the short code [calbtn] in your page content ro you can call it custom templates php code.
